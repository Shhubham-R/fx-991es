# Casio fx-991ES PLUS (2nd Edition) Emulator

A pixel-perfect, highly responsive behavioral replica of the iconic **Casio fx-991ES PLUS (2nd Edition)** scientific calculator on the web. Built with raw HTML, CSS, and modern JavaScript, it replicates the natural textbook display (Natural V.P.A.M.) and supports major scientific calculation functions.

## 🚀 Live Demo
Visit the emulator online here: **[Live Emulator Link](https://shhubham-r.github.io/fx-991es/)**

---

## 🎨 Key Features
- **Natural V.P.A.M. (Natural Textbook Display):** Displays expressions (fractions, powers, roots, etc.) exactly as they appear in textbooks.
- **417 Scientific Functions:**
  - Multi-play / Replay functionality.
  - 40 Scientific Constants (CODATA 2018 values) e.g., proton mass, Planck constant.
  - 40 Metric Unit Conversions.
  - Equation Solver, Matrix & Vector calculations.
  - Table and STAT modes.
- **Dynamic LCD Display:** Pixelated font styling, setup status indicators (S, A, M, STO, RCL, D/R/G, Math), and active cursors.
- **Modern Responsive Design:** Glassmorphic backgrounds, subtle keypress feedback, responsive scaling for mobile devices, and solar panel visual accents.
